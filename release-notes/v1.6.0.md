# LDC Construction Tools v1.6.0

**Release Date:** December 16, 2025

## Phase 4: Project Schedule Builder

This release introduces a comprehensive project scheduling system with Gantt-style calendar visualization.

### New Features

#### Project Schedule Management
- **Schedule Versioning** - Create multiple schedule versions per project (e.g., "Initial", "Revised December", "Final")
- **Duplicate Versions** - Copy an existing schedule to create a new version with all entries
- **Set Current Version** - Mark a version as the "current" official schedule
- **Delete Versions** - Remove outdated schedule versions (cannot delete current version)

#### Gantt-Style Calendar View
- **Continuous Bars** - Schedule entries display as horizontal bars spanning their date range
- **Trade Team Icons** - Visual icons for each trade team (‚ö° Electrical, üè† Exteriors, etc.)
- **Color-Coded Entries** - Each trade team has distinct background colors
- **Note Indicators** - üìù emoji appears on entries with notes
- **Monthly Navigation** - Browse schedules by month with prev/next controls

#### Schedule Entry Management
- **Trade Team & Crew Assignment** - Assign entries to specific trade teams and crews
- **Date Range Selection** - Set start and end dates for each entry
- **Volunteer Count** - Track expected volunteer counts per entry
- **Notes Field** - Add detailed notes with full display in entry list

#### Calendar Page as Central Hub
- **Project Selector** - Choose any project from dropdown
- **URL Parameter Support** - Link directly to a project's schedule via `/calendar?project=ID`
- **"View Schedule" Integration** - Project detail page links directly to calendar

#### Excel Export
- **Calendar View Sheet** - Gantt-style layout with dates across columns
- **Detailed List Sheet** - Full table with all entry fields including notes
- **Auto-calculated Duration** - Days between start and end dates

### Technical Details
- New Prisma models: `ProjectScheduleVersion`, `ProjectScheduleEntry`
- Full CRUD API routes for schedule versions and entries
- Shared trade team configuration (`trade-team-config.ts`)
- XLSX library for Excel export

### Database Migration
Run `npx prisma migrate deploy` to apply the new schedule tables.
