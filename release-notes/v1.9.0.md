### üìä Complete Import & Export System

We've added comprehensive import and export capabilities across the entire application, making it easier to manage large amounts of data and create reports.

**What you can do:**
- Bulk import volunteers and projects using CSV files
- Export data to Excel, PDF, or CSV formats
- Generate professional reports for meetings and documentation
- Back up your data for safekeeping
- Share information with team members

**How to use it:**

**Importing Data:**
1. Click the "Template" button to download a CSV template
2. Fill in your data using Excel or Google Sheets
3. Save as CSV format
4. Click "Upload CSV" and select your file
5. The system will show you how many records were successfully imported

**Exporting Data:**
1. Apply any filters to narrow down your data (optional)
2. Click the green "Export" button
3. Choose your format: Excel, PDF, or CSV
4. The file downloads automatically with a clean, timestamped name

**Available on:**
- **Volunteers Page:** Import/Export volunteers with full details
- **Projects Page:** Import/Export project information
- **Calendar Page:** Export schedules in Excel or PDF (with duration calculations)
- **Trade Teams Overview:** Export organizational structure
