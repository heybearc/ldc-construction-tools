### Secondary Role System for Better Organization

We've added a new way to represent your organizational structure within LDC Tools!

**What's new:**
- Users can now have both a **System Role** (for permissions) and an **LDC Role** (for organizational position)
- Personnel Contacts and other LDC roles are now properly represented in the system
- Improved crew request assignment filtering to include users with organizational roles

**What you can do:**
- Administrators can assign LDC Roles to users through the User Management page
- Users with Personnel Contact LDC roles now appear in crew request assignment dropdowns
- Better visibility of organizational structure within Construction Groups

**How to use it:**
1. Go to **Admin** ‚Üí **Users**
2. Click **Edit** on any user
3. Set their **LDC Role** (e.g., Personnel Contact, Zone Overseer)
4. Set their **Construction Group** (e.g., CG 01.12)
5. Click **Update User**

**Learn more:** Visit **Help** ‚Üí **Admin Panel** ‚Üí **Understanding User Roles**

### Crew Request Assignment Visibility

You can now see who's working on each crew request!

**What's new:**
- Each crew request now shows "Assigned to: [Name]" below the request details
- Personnel team members can easily see workload distribution
- Better tracking of who's handling each request

**What you'll see:**
- On the Manage Requests page, look for the blue "Assigned to:" text under each request
- This appears for all requests that have been assigned to a team member

---

### Construction Group Selection Improvements

Setting up users is now easier with improved Construction Group selection.

**What's new:**
- Construction Group dropdown now shows readable codes like "CG 01.12"
- No more confusing database IDs
- Easier to assign users to the correct Construction Group
