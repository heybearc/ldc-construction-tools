# Release Notes - v1.8.0

**Release Date:** December 18, 2025

## New Features

### Supporting Congregations Management
Track and manage congregations that support your construction projects.

#### Congregations Master List
- **New Congregations Page** - Manage all congregations in one place
- Store congregation details: Name, State, Congregation Number
- Track coordinator information: Name, Phone, Email
- Store congregation email for group communications
- Search and filter congregations
- View statistics: total congregations, coordinators, states represented

#### Project-Specific Congregation Assignments
- **Assign congregations to projects** with project-specific contacts
- **Food Contact** - Track who handles food arrangements for each congregation
- **Volunteer Contact** - Track who coordinates volunteers from each congregation
- **Security Contact** - Track who handles security responsibilities
- Each contact includes: Name, Phone, Email
- Add notes for special instructions or information
- Easy add/edit/remove functionality

### Navigation Update
- Added "Congregations" link to the main navigation bar

## Database Updates
- Added `Congregation` model for master congregation data
- Added `ProjectCongregationAssignment` model for project-specific assignments
- New tables: `congregations`, `project_congregation_assignments`

## Technical Details
- New API endpoints:
  - `GET/POST /api/v1/congregations` - List and create congregations
  - `GET/PATCH/DELETE /api/v1/congregations/[id]` - Manage individual congregations
  - `GET/POST /api/v1/projects/[id]/congregations` - Project congregation assignments
  - `PATCH/DELETE /api/v1/projects/[id]/congregations/[assignmentId]` - Manage assignments
