### üìã Multi-Tenant Audit Logs

Track all Construction Group operations and cross-tenant activities with our new comprehensive audit logging system.

**What you can do:**
- See who created, edited, or deleted Construction Groups
- Track when Super Admins switch between CGs
- Monitor user and volunteer changes across all CGs
- View detailed before/after values for all changes
- Export audit logs to CSV for reporting

**How to use it:**
1. Go to **Admin** ‚Üí **Audit Logs**
2. Click on **Multi-Tenant Audit Logs**
3. Use filters to find specific activities
4. Click **Show** on any entry to see full details
5. Export to CSV when needed

**Who can use it:** Super Admins only

### ‚úèÔ∏è Construction Group Management

Super Admins can now edit and delete Construction Groups directly from the Organization page.

**What you can do:**
- **Edit CG Details** - Update CG codes or names
- **Delete CGs** - Safely remove CGs (soft delete preserves data)
- **Reactivate CGs** - Restore accidentally deleted CGs
- **Track Changes** - All operations logged in audit logs

**How to use it:**
1. Go to **Admin** ‚Üí **Organization**
2. Find the Construction Group in the tree
3. Click **Edit** (pencil icon) or **Delete** (trash icon)
4. Confirm your changes

**Important:** Deleting a CG doesn't remove data - it's a soft delete that can be reversed.

**Who can use it:** Super Admins only
