# Excel Dashboard Formulas for LDC Tools POC

## Dashboard Layout

Create a new sheet named "Dashboard" with the following sections:

---

## SECTION 1: SUMMARY STATISTICS (Rows 1-10)

### Cell A1: Title
="CONSTRUCTION GROUP 0205 - VOLUNTEER DASHBOARD"
(Format: Bold, 18pt, Centered across A1:F1)

### Cell A3: Total Volunteers
="Total Volunteers:"
### Cell B3: Count
=COUNTA(Volunteers[FirstName])

### Cell A4: Active Projects
="Active Projects:"
### Cell B4: Count
=COUNTIF(Projects[Status],"Active")

### Cell A5: Pending Crew Requests
="Pending Crew Requests:"
### Cell B5: Count
=COUNTIF(CrewRequests[Status],"Pending")

### Cell A6: Total Assignments
="Total Assignments:"
### Cell B6: Count
=COUNTA(Assignments[AssignmentID])

## SECTION 4: ACTIVE PROJECTS SUMMARY (Rows 30-40)

### Cell A30: Header
="ACTIVE PROJECTS"
(Format: Bold, 14pt)

### Cell A31-E31: Column Headers
A31: Project Name
B31: Location
C31: Status
D31: Assigned Workers
E31: Project Manager

### Rows 32-36: Use FILTER formula (Excel 365)
A32: =FILTER(Projects[ProjectName],Projects[Status]="Active","No active projects")
B32: =FILTER(Projects[Location],Projects[Status]="Active","")
C32: =FILTER(Projects[Status],Projects[Status]="Active","")
D32: Formula to count assignments per project (see below)
E32: =FILTER(Projects[ProjectManager],Projects[Status]="Active","")

### Cell D32: Count Assignments for Project
=COUNTIF(Assignments[ProjectName],A32)
(Copy down to D33:D36)

## CONDITIONAL FORMATTING

### Crew Requests Status (Column E44:E53)
- Pending = Yellow fill
- Approved = Light Blue fill
- Fulfilled = Green fill
- Cancelled = Red fill

### Priority (Column F44:F53)
- High = Red text, bold
- Medium = Orange text
- Low = Gray text

---

## DATA VALIDATION SETUP

### For Volunteers Sheet:
1. TradeTeam column: List from TradeTeams[TeamName]
2. Status column: List = Active, Inactive, On Leave
3. Availability column: List = Full-time, Part-time, Weekends
4. Congregation column: List = Smithville, Riverside, Oakdale

### For Assignments Sheet:
1. VolunteerName column: List from Volunteers (FirstName + LastName)
2. ProjectName column: List from Projects[ProjectName]
3. TradeTeam column: List from TradeTeams[TeamName]
4. Role column: List = Worker, Crew Lead, Supervisor
5. Status column: List = Assigned, Confirmed, Completed, Cancelled

### For CrewRequests Sheet:
1. ProjectName column: List from Projects[ProjectName]
2. TradeTeamNeeded column: List from TradeTeams[TeamName]
3. Status column: List = Pending, Approved, Fulfilled, Cancelled
4. Priority column: List = High, Medium, Low

## ADVANCED FEATURES (Optional)

### Dynamic Project List (Excel 365)
Use FILTER function to show only active projects:
=FILTER(Projects,Projects[Status]="Active","No active projects")

### Available Volunteers Report
Create new sheet "Available" with formula:
=FILTER(Volunteers,(Volunteers[Status]="Active")*(COUNTIF(Assignments[VolunteerName],Volunteers[FirstName]&" "&Volunteers[LastName])=0),"No available volunteers")

### Volunteer Lookup
Create dropdown to select volunteer and show all their assignments:
Cell A1: Dropdown with volunteer names
Cell A3: =FILTER(Assignments,Assignments[VolunteerName]=A1,"No assignments")
