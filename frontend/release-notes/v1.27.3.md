# What's New in Version 1.27.3

**Released:** February 15, 2026

---

## ‚ú® New Features

### Password Reset Functionality

**Admin-initiated password reset for users**
- Admins can now send password reset emails to users from the user management page
- Users receive a secure reset link via email
- Password reset page with secure token validation
- 24-hour expiration on reset tokens

**How it works:**
1. Admin clicks "Reset Password" on a user's profile
2. User receives email with reset link
3. User clicks link and sets new password
4. Token is automatically deleted after use

### D-024 Feedback Management Standard

**Feedback ID System (FB-001 format)**
- All feedback items now have sequential IDs: FB-001, FB-002, FB-003, etc.
- Matches TheoShift implementation for consistency
- User-facing IDs for easy reference and tracking

**Resolution Comments**
- All feedback items now include resolution comments
- Tracks how and when issues were resolved
- Provides audit trail for feedback management
- Complies with D-024 Unified Feedback Management Standard

**Feedback Comment Permissions**
- Fixed: Non-admin users can now add comments to their own feedback
- Added missing API endpoint for feedback submitters
- **Commits:** 61569ae, cf29d99
