# Release Notes - v1.5.0

**Release Date:** December 16, 2024

## Phase 3: Project Management

This release introduces comprehensive project management capabilities, allowing Construction Groups to track projects, assign crews, and link to external resources.

### New Features

#### Projects List Page
- **Project Dashboard** - View all projects with status statistics (Total, Active, Planning, Completed)
- **Search & Filter** - Search projects by name, location, or project number; filter by status
- **Quick Create** - Create new projects directly from the list page

#### Project Details
- **Project Information** - Track project number, location, type, and current construction phase
- **Project Types** - Kingdom Hall, Assembly Hall, Remote Translation Office, Branch Office, Bethel, Other
- **Construction Phases** - Site Preparation, Mobilization, Site Work, Structural, Rough-in, Finishes, Final Prep
- **Timeline Tracking** - Start and end dates with visual timeline display
- **Description/Notes** - Free-form text area for project details and notes

#### External Links Integration
- **JW SharePoint URL** - Direct link to project documents on SharePoint
- **Builder Assistant URL** - Direct link to project in Builder Assistant (ba.jw.org)
- **One-Click Access** - Clickable buttons to open external resources in new tab

#### Crew-to-Project Assignments
- **Assign Crews** - Assign trade team crews to projects
- **View Assignments** - See all crews assigned to a project with member counts
- **Remove Assignments** - Easily remove crews from projects when work is complete

### Technical Improvements
- New Prisma schema fields for enhanced project data
- Next.js API routes for project CRUD operations
- Crew assignment API endpoints
- CG-scoped data access (multi-tenant)

### Database Changes
- Added projectNumber, location, projectType, currentPhase fields to Project model
- Added jwSharepointUrl, builderAssistantUrl fields for external links
- Added ProjectCrewAssignment model for crew-to-project tracking
