### Edit Active Requests
You can now edit crew change requests before they're completed! No more deleting and resubmitting when you need to make a small correction.

**What you can do:**
- Fix typos in volunteer names
- Update BA ID numbers
- Change crew or project assignments
- Modify comments and notes

**How to use it:**
1. Go to **Manage Requests** page
2. Find a request with NEW or IN_PROGRESS status
3. Click the **Edit** button
4. Make your changes
5. Click **Save Changes**

---

### Clone Completed Requests
Need to submit a similar request? Clone a completed request to save time!

**What you can do:**
- Reuse request details for new submissions
- Quickly submit similar crew changes
- Modify cloned details before submitting

**How to use it:**
1. Find a completed request
2. Click the **Clone** button
3. Review and modify the pre-filled form
4. Submit as a new request

---

### Reopen Completed Requests
Made a mistake? Need to undo completed work? You can now reopen requests!

**What you can do:**
- Change completed requests back to IN_PROGRESS
- Add explanation for why it's being reopened
- Maintain full audit trail

**How to use it:**
1. Find a completed request
2. Click the **Reopen** button
3. Enter a reason for reopening
4. Request returns to IN_PROGRESS status

---

### Regional Staff Management
Administrators can now manage Zone and Region level oversight roles directly in the system.

**What you can do:**
- Assign Zone Overseers (ZO) and Assistants (ZOA)
- Assign Regional Overseers (RO) and Assistants (ROA)
- View all regional staff from Construction Group detail page
- Track oversight hierarchy across zones and regions

**How to use it:**
1. Go to **Admin â†’ Organization**
2. Select a Construction Group
3. Scroll to **Regional Staff** section
4. Click **+ Add** to assign roles
