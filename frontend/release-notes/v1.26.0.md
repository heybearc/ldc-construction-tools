### Dynamic Dashboard Region Display

The dashboard now automatically shows the correct region based on your Construction Group selection.

**What you can do:**
- See your region name displayed prominently on the dashboard
- Super Admins can switch between Construction Groups and see the region update automatically
- When viewing "All Construction Groups", the dashboard shows "All Regions"

**How it works:**
1. The dashboard automatically detects your Construction Group
2. Your region name appears under "Welcome to LDC Tools"
3. Super Admins: Use the Construction Group selector in the top navigation to switch between groups
4. The region display updates immediately after switching

**Who benefits:**
- **Super Admins**: Easily see which region you're currently viewing when switching between Construction Groups
- **All Users**: Clear confirmation of which region's data you're viewing
