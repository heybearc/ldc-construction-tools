### Crew Request Submission

**Fixed: Submit on behalf of others functionality**
- Personnel Contacts can now successfully submit crew requests on behalf of volunteers
- Fixed role permission checks to properly query the `volunteer_roles` table
- Resolved issue where PC/PCA/PC_SUPPORT roles were not being recognized correctly
- **Commit:** 6f4593f

- When Personnel Contacts tried to submit crew requests for other volunteers, the system would fail
- Role code mismatch between PC_SUPPORT and PC-Support caused permission errors

**What's fixed:**
- Role permission checks now properly query active PC/PCA/PC_SUPPORT roles
- Submit-on-behalf functionality works as expected
- Personnel Contacts can manage crew requests for their volunteers
