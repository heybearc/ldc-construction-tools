**Released:** December 17, 2025

## ‚ú® New Features

### Trade Team Oversight Assignment
Trade Team Overseers can now be directly assigned to a trade team from the Volunteers page.

**What you can do:**
- Assign a volunteer as Trade Team Overseer to a specific trade team
- The assignment is saved and displayed correctly in the volunteer directory

**How to use it:**
1. Go to Volunteers page
2. Edit a volunteer
3. Select "Trade Team Overseer" as the role
4. Choose the trade team from the dropdown
5. Save

---

### User-Volunteer Linking
Link platform user accounts to volunteer records for seamless login access.

**What you can do:**
- Connect a volunteer to their platform login account
- See which volunteers have linked accounts

**How to use it:**
1. Go to Volunteers page
2. Edit a volunteer
3. Scroll to "Link User Account" section
4. Select the user account from the dropdown
5. Save

---

### Oversight Status Indicators
Trade Teams page now shows at-a-glance oversight status for each team.

**What you'll see:**
- **TTO badge**: Shows if Trade Team Overseer is assigned (green = filled, red = missing)
- **TTOA badge**: Shows assistant positions filled (0/2, 1/2, 2/2)
- **Crews needing TCO**: Warning showing how many crews need a Trade Crew Overseer
